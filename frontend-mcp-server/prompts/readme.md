# Prompt

```md
---
name: 项目总结
description: 对当前项目进行整体概览和技术总结
arguments:
  - name: 受众
    description: 阅读总结的目标人群
    default: 前端开发者
---

你是一名经验丰富的前端架构师。

请基于当前仓库内容，对项目进行总结，要求包括：

1. 项目的核心目标和业务背景
2. 使用的主要技术栈（如框架、构建工具等）
3. 关键目录结构及其职责
4. 项目中值得注意的设计点或约定

目标受众：**{{受众}}**

请使用清晰的分点说明，必要时引用仓库中的文件或目录。

```

```md
---
name: 错误排查
description: 帮助分析并解决代码或运行时错误
arguments:
  - name: 错误信息
    description: 请粘贴完整的错误日志或报错信息
    required: true
---

你是一名擅长定位问题的高级工程师。

下面是遇到的错误信息：

{{错误信息}}

请按以下步骤分析：

1. 判断错误类型（编译期 / 运行期 / 环境问题等）
2. 推测最可能出问题的代码位置
3. 给出修复思路或示例代码
4. 提醒可能的副作用或回归风险
```
